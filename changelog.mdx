## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature","improvement"]}>

## New Features

- Added webhook support for real-time integrations with external tools like Slack and GitHub.
- Introduced space archiving to preserve old documentation without cluttering active lists.

## Improvements

- Enhanced page editor with better markdown preview and syntax highlighting for 10+ languages.
- Optimized search functionality to include full-text indexing across all spaces.

## Bug Fixes

- Fixed permission inheritance issues when moving pages between spaces.
- Resolved export errors for pages containing embedded images.

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking","feature"]}>

## New Features

- Launched collaborative editing with real-time cursors and conflict resolution.
- Integrated version history with diff views and easy revert options.

## Breaking Changes

- Updated API endpoints to use `/v2/` prefix; migrate existing integrations.
- Changed default permissions for new spaces to require explicit member invites.

## Improvements

- Improved mobile responsiveness for the dashboard and page viewer.
- Added keyboard shortcuts for common actions like bold (`Ctrl + B`) and link insertion (`Ctrl + K`).

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature"]}>

## Initial Release

- Core space and page creation with rich text and markdown editors.
- Basic user authentication and role-based permissions (admin, editor, viewer).
- Simple search and export features for individual pages.

## Improvements

- Dashboard overview with quick access to recent spaces and pages.
- Notification system for mentions and updates within spaces.

</Update>
